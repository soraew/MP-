# MP

数理最適化アルゴリズムの実装

#### branching_class.py -> 分枝限定法

#### dp.py -> 動的計画法
usage :

if objective function : 
7x_1 + 8x_2 + x_3 + 2x_4

and constraint : 
4x_1 + 5x_2 + x_3 + 3x_4 <= 6

  ```shell
  >python dp.py
  Enter number of variables : 4 

  Enter objective function array : 7 8 1 2

  Enter constraint function array : 4 5 1 3

  Enter constraint : 6```
